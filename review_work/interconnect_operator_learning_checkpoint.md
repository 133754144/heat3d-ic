# 异构热导率互联结构专题检查点

更新时间：2026-04-14

## 专题问题

如何在热仿真的算子学习框架中处理含有不同热导率的导线互联结构？

这里的“导线互联结构”主要包括：

- TSV / TTSV
- bump / micro-bump
- 互连金属与介质复合层
- chiplet 层间插层与界面结构

## 当前得到的核心判断

### 1. 这个问题的难点不只是“材料不同”

真正困难在于以下四点同时出现：

1. 热导率强跳变：高导热金属与低导热介质并存；
2. 多尺度：细导线、超薄绝缘层与大尺度芯片区域同时存在；
3. 界面条件：温度连续与热流连续约束必须被正确处理；
4. 几何变化：互联阵列密度、尺寸、位置和层间结构可能变化。

### 2. 现有文献中的主要处理策略

#### A. 显式导热率场编码

- 代表：DeepOHeat
- 做法：把 `full-chip flexible material conductivity distribution` 作为输入配置之一，与功率图、边界条件共同输入算子学习模型。
- 意义：适合将材料分布视为连续场或离散网格场直接学习。

#### B. 等效热导率/均匀化

- 代表：PI-ONet、Fast Thermal Modeling of TTSV via Bounded Neural Networks
- 做法：先对 TSV / bump / 插层等复杂细结构提取等效热导率，再把它们作为均匀等效层纳入更大尺度热分析。
- 意义：这是当前处理细互联结构最现实、最常见的方法。

#### C. 界面条件显式入损失

- 代表：PI-ONet
- 做法：在物理损失中显式写入层间界面上的温度连续与热流连续条件。
- 意义：对于不同热导率材料接触问题非常关键。

#### D. 几何感知与图算子

- 代表：GINO、Adaptive Graph Learning
- 做法：使用点云/SDF/图结构处理不规则几何与复杂界面变化。
- 意义：适合未来从“均匀化层”进一步走向“显式复杂互联几何”的方向。

#### E. 多保真与混合求解

- 代表：ARO、T-Fusion、DeepOHeat-v1、MFIT
- 做法：结合低保真等效模型与少量高保真显式模型，或将算子学习结果交给传统求解器局部修正。
- 意义：适合处理含精细互联结构时数据昂贵、局部梯度陡峭的问题。

### 3. 当前领域的明显空白

1. 多数 3D IC 热仿真算子学习论文还没有真正显式建模细粒度 BEOL/导线网络。
2. 很多工作处理的是层级等效材料，而不是显式互联几何。
3. 对“导热率张量场 + 界面跳跃条件 + 任意几何”的统一算子学习框架仍不成熟。
4. 目前最现实的路线仍然是“等效建模 + 物理约束 + 局部高保真修正”。
