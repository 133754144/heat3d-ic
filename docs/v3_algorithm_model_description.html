稳态热场算子
模型构成与计算链路

V3 使用多尺度 RIGNO，将坐标、导热系数、热源和边界条件映射为固定物理点云上的温升场，再恢复为绝对温度。

RIGNO p2r · r2r · r2p processor ×6 normalized-ΔT MSE source 11e9d2f

一句话定义

(x, k, q, BC) → T(x)。模型先预测归一化温升，再通过 train-only 统计量反归一化并加回参考温度。

ΔT = T − T_ref   →    ΔT̂_norm   →    T̂ = T_ref + ΔT̂

它解决什么问题？

在同一固定三维点云上，学习异质材料、局部热源与边界条件对稳态温度分布的联合影响。它是监督式算子回归，不是 PDE solver。

02 · Architecture

模型构成

每一层只承担一个清晰的空间变换：物理节点 → 区域 latent → 区域传播 → 物理节点。

输入与归一化

监督目标是温升，不把 target-derived feature 输入网络。

  • x：三维坐标
  • k=(k_x,k_y,k_z)：diag3
  • q：局部热源；BC：相对边界条件
  • μ / σ 只由 train 拟合

多尺度图

物理节点保留输出分辨率，区域节点提供更大的 receptive field。

  • Vp：physical nodes（medium 数据为 1024 点）
  • Vr：regional nodes
  • p2r / r2r / r2p 三类 edge set
  • discrete physical coverage，最小覆盖 ≥ 1

Encoder · p2r

把物理节点上的材料、热源和边界信息聚合到区域 latent。

(h_p⁰, h_r⁰) = Encoder(p2r, x, k, q, BC)

输出并保留 latent_pnodes 与 latent_rnodes。

Processor · r2r × 6

只在区域图上进行六次消息传播，让局部影响跨区域扩散。

h_rℓ+1 = Processor(h_rℓ, Er→r)

每步包含 edge/node MLP、归一化和残差更新。

Decoder · r2p

将处理后的 regional latent 与初始 physical latent 合并，恢复每个物理点的输出。

ŷ_norm = Decoder(h_r⁶, h_p⁰, Er→p)

结果是归一化温升场，不是绝对温度。

恢复与评估

反归一化后加回参考温度，再计算 raw 与 relative 诊断。

ΔT̂ = σ_train · ŷ_norm + μ_train
T̂ = T_ref + ΔT̂

报告 RMSE、MAE、amplitude、correlation、hotspot 和 strong-q。

03 · Formal path

计算逻辑

图上的一次前向传播

对一个样本，先构造三类边，再按顺序执行 encoder、processor 和 decoder。聚合使用邻居消息的 segment mean，使不同节点的边数不会直接改变量纲。

G = (Vᵖ, Vʳ, Eᵖ→ʳ, Eʳ→ʳ, Eʳ→ᵖ)
ŷ_norm(xᵢ) = Decoder(Processor(Encoder(x, c)))ᵢ
# train-only statistics
stats = fit_stats(train)
for sample in dataset:
    x, c, delta_t = load(sample)
    x_n, c_n = normalize(x, c, stats)
    y_n = normalize(delta_t, stats)

    G = build_graph(x_n, policy="discrete_physical_coverage")
    h_p, h_r = encoder(G.p2r, x_n, c_n)
    for step in range(6):
        h_r = processor(G.r2r, h_r)
    y_hat_n = decoder(G.r2p, h_r, h_p)

    loss = mse(y_hat_n, y_n)
    update_adamw(loss)

delta_t_hat = inverse_normalize(y_hat_n, stats)
temperature_hat = t_ref + delta_t_hat

05 · Interpretation

模型边界

这是监督式算子回归

V3 没有 PDE residual、界面通量连续性、能量守恒或独立的 Dirichlet/Neumann penalty。它学习的是合成数据分布上的 coefficient-to-temperature 映射。

指标的正确读法

valid_base_mse 位于归一化空间，不能直接写成真实 relative RMSE。正式比较需要另外报告 raw RMSE、真实 RMS-relative、sample-first 与 point-global 聚合。
